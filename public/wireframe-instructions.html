Как экспортировать данные и восстановить информацию

template
1

Все данные хранятся в облачной PostgreSQL базе. Экспортируй данные в JSON для резервной копии:

Скачать данные 
// Открой в браузере
/api/download-db

// Файл скачается автоматически
wiki-codex-backup-YYYY-MM-DD.json

// Содержит:
  - документы
  - заметки
  - инструкции
  - термины, категории, теги
2

Создай моментальный снимок данных через API:

API бэкапа 
POST /api/backup

// Вернёт статистику экспортированных данных
Скачать JSON 
GET /api/download-db

// Скачает полный JSON-экспорт всех данных
3

Данные защищены управляемой PostgreSQL базей. Для восстановления требуется доступ к провайдеру.

Vercel Postgres 
// Данные автоматически сохраняются
// Бэкапы управляются провайдером
Neon/Supabase
4

После развертывания открой приложение и проверь:

Что проверить 
1. Sidebar -- счётчики не нули
2. Документы -- открой любой документ
3. Заметки -- есть ли твои заметки
4. Инструкции -- встроенные шаблоны видны
5. Footer -- "Built with: Next.js 16 + TypeScript + Tailwind CSS"
